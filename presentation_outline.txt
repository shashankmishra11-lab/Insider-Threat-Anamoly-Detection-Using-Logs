Technical Appendix - Libraries, Tools & Functions

1. Frontend Libraries
   - Bootstrap 5: Frontend framework for responsive design
   - Boxicons: Modern icon library for UI elements
   - Chart.js: Interactive data visualization library
   - LottieFiles: Lightweight animation library for loading states

2. Backend Libraries (Python)
   - Flask: Web framework for backend API
   - SQLite3: Database management
   - Pandas: Data manipulation and analysis
   - NumPy: Numerical computing and array operations
   - Scikit-learn: Machine learning algorithms for anomaly detection

3. Key Functions & Components

   a) Flask Routes
      - /upload: Handles file upload and processing
      - /analyze: Processes log data and detects anomalies
      - /stats: Returns real-time statistics
      - /api/anomalies: RESTful endpoint for anomaly data

   b) Data Processing
      - preprocess_logs(): Cleans and formats input data
      - detect_anomalies(): Implements anomaly detection algorithm
      - calculate_risk_score(): Assigns risk levels to activities

   c) Database Operations
      - save_anomaly(): Stores detected anomalies
      - get_stats(): Retrieves statistical summaries
      - update_risk_levels(): Updates user risk profiles

   d) Frontend Functions
      - initializeCharts(): Sets up data visualizations
      - updateStats(): Real-time dashboard updates
      - handleFileUpload(): Manages file upload UI/UX
      - renderAnomalyTable(): Displays anomaly data

4. Development Tools
   - VS Code: Primary IDE
   - Git: Version control
   - Chrome DevTools: Frontend debugging
   - SQLite Browser: Database management

5. Key CSS Features
   - CSS Variables: Dynamic theming
   - Flexbox/Grid: Responsive layouts
   - Glass Morphism: Modern UI effects
   - CSS Animations: Smooth transitions

6. JavaScript Components
   - Fetch API: Asynchronous data requests
   - Event Listeners: Interactive UI elements
   - DOM Manipulation: Dynamic content updates
   - WebSocket: Real-time data updates

7. Security Implementations
   - Input Sanitization: prevent XSS attacks
   - CSRF Protection: secure form submissions
   - File Validation: secure upload handling
   - Error Handling: graceful error management
